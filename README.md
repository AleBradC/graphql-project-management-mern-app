# Project Title

Management MERN app

# Dependencies

- GraphQL
- Express.js
- MongoDB
- Mongoose
- React
- Apollo Client

# Description

Simple app where you can add a client or delete the client, then you can create a project, view, update or delete.

# Getting Started

- clone the project
- npm run dev - to start the server
- cd client
- npm run start - to start the client
